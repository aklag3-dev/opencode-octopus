# Octopus Venture Engine

First metaskill for Octopus. One recipe (MICRO_TOOL), two modes (hunt/harvest), one compounding intelligence loop.

## Philosophy
Build small, sell small, learn fast. The goal is not a unicorn — it's 10 micro-tools each generating $100-500/mo, compounding through intelligence.

## Prerequisites
- Cloudflare account with wrangler CLI installed
- Lemon Squeezy account with API key
- `.octopus/.env` with credentials
- `.octopus/venture/` and `.octopus/intelligence/` directories

## Modes
- `cycle hunt [niche]` — Discover opportunity, build micro-tool, launch, evaluate
- `cycle harvest [product-name]` — Compound a winner with SEO, conversion, variants
- `cycle status` — Check all active products and their metrics
