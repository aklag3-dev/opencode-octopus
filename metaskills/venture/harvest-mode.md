# Harvest Mode

Triggered by: `cycle harvest [product-name]`

## Sprint Types

### Sprint 1: SEO Expansion (Researcher + Marketer)
- Find 10 long-tail keywords
- Create supporting content for each
- Update meta tags and structure

### Sprint 2: Conversion Optimization (Designer + Maker)
- One A/B test per sprint
- Improve conversion funnel
- No feature additions

### Sprint 3: Variant Expansion (conditional, >20 purchases)
- Create 1-3 spin-off variants
- Target adjacent use cases
- Share codebase with original

### Sprint 4: Bundle/Upsell (conditional, 2+ products in same niche)
- Create bundle offer
- Set up cross-sell links

## Cadence
- Every 14 days: run next sprint
- Evaluate: KILL if no growth for 3 consecutive sprints
- Maintain: weekly SEO content only
