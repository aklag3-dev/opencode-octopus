# Hunt Mode

Triggered by: `cycle hunt [niche]`

## Phases

### Phase 1: DISCOVER (Yellow Researcher)
- Generate 10 candidate micro-tool ideas in the niche
- Score each on 7 factors (pain intensity, search volume, build complexity, monetization potential, competition, defensibility, personal fit)
- Weighted scoring model applied
- Return top 3 candidates with scores

### Phase 1.5: CHAIRMAN REVIEW (mandatory first 10 cycles)
- Present top candidate to human for approval
- Walk through falsification checklist
- If rejected, move to next candidate

### Phase 2: SPEC (Red-Orange Designer)
- Single-page tool design
- ASCII wireframe
- Technical architecture
- Build spec for Maker (~30 min)

### Phase 3: BUILD + COPY (Blue Maker + Green Marketer, parallel)
- Maker builds the tool (~2-3 hrs)
- Marketer writes copy, meta tags, social assets

### Phase 4: LAUNCH (~15 min)
- Deploy to Cloudflare Pages
- Configure Lemon Squeezy payment
- Submit to directories

### Phase 5: EVALUATE
- Day 3: traffic check
- Day 7: conversion check
- Day 14: revenue check
- Day 30: KILL/PIVOT/SCALE decision
