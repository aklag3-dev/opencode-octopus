# Product Types

## Active: MICRO_TOOL

### Definition
Single-page web tools that solve one specific problem. Examples: URL encoder, JSON formatter, color converter, markdown previewer.

### Build Constraints
- Single HTML file (all inline)
- Vanilla JS (no frameworks)
- Deployed to Cloudflare Pages
- Monetized via Lemon Squeezy (pay-what-you-want or fixed $3-9)
- SEO-first: unique title, meta description, structured data

### Evaluation Timeline
| Day | Check | Action |
|-----|-------|--------|
| 3 | Traffic > 0 | If zero, investigate indexing |
| 7 | > 50 visitors | If not, improve SEO |
| 14 | > 1 purchase or > 200 visitors | If neither, consider PIVOT |
| 30 | > $50 revenue | If not, KILL |

### Kill Criteria (any one triggers KILL)
- Zero organic traffic after 14 days
- Zero purchases after 30 days
- Build cost exceeds $50 (API costs, etc.)
