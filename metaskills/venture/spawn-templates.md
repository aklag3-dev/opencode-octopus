# Venture Mode Spawn Templates

## Purple Manager Venture Commands
When user says `cycle hunt [niche]`, `cycle harvest [product]`, or `cycle status`:
- Read hunt-mode.md / harvest-mode.md before dispatching any agent
- Follow the fixed pipeline exactly
- Do not skip the Chairman Review phase (first 10 cycles)

## Yellow Researcher (Hunt Mode)
Extends the standard Researcher template with:
- Generate exactly 10 candidates
- Score each on all 7 factors with explicit values
- Weighted scoring: Pain (25%), Search Volume (20%), Build Complexity (15%), Monetization (15%), Competition (10%), Defensibility (10%), Personal Fit (5%)
- Format: table with scores + 2-paragraph rationale per candidate

## Red-Orange Designer (Hunt Mode)
Extends the standard Designer template with:
- Design for a single HTML file (all inline)
- Maximum 3 screens/views
- Mobile-first responsive
- No login/auth required
- Payment via Lemon Squeezy overlay (not a separate page)

## Blue Maker (Hunt Mode)
Extends the standard Maker template with:
- Single HTML file output
- All CSS and JS inline
- Deploy to Cloudflare Pages
- Lemon Squeezy purchase button integrated
- Google Analytics (or Plausible) single-line embed
- No build step, no npm dependencies

## Green Marketer (Hunt Mode)
Extends the standard Marketer template with:
- SEO-first: unique title (< 60 chars), meta description (< 160 chars), structured data JSON-LD
- Social cards: Twitter/X card, Open Graph
- One landing page variant
- Distribution: 1 directory submission, 1 forum post, 1 social share

## Yellow Analyst (Harvest Mode)
Extends the standard Analyst template with:
- Compare current metrics against previous sprint
- Determine if growth rate is positive, flat, or negative
- Recommend: continue, pivot, or kill
